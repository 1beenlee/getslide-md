# TrailNote

TrailNote is an offline-first PWA for campus hiking clubs. Members save a route, checkpoint notes, and a packing list before leaving reception. A service worker caches the route package so the same route remains available without a connection.

For the demo, create a route while online, turn on airplane mode, then reopen the saved route. Repository: https://example.com/repository

The project was built for a 24-hour campus hackathon final demo.
